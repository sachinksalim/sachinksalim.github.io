1. Data
  Data is found in data/ folder.
  data.tsv and family.tsv are tab-separated-value files.
  Contents in data.tsv are self explanatory.
  In family.tsv, for each new family, i.e., for each new webpage, add a row with their address followed by the parents' IDs and then the children's IDs.
  These files can be open in MS Excel for easy editing.
  
  Run the python file parse.py inside data/ folder to generate the JavaScript(.js) files using the new data.
  
2. Images
  Images should be saved in the folder images/ with name as <id>.jpg where id can be found from data.tsv file.
  
3. CSS
  print.css has the style of an A4 sheet.
  web.css can be modified to be in any size.
